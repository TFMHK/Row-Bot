# שו"ב בדפדפן

ממשק שו"ב זה עובד מול שרת Python מקומי על המחשב. השרת מחובר ב-USB לארדואינו החוף, והדפדפן בטלפון או במחשב מדבר אל השרת ב-HTTP.

## מה כולל הממשק

- תמונת מכ"ם ב-Canvas (מבוססת טלמטריה `usRadar` + `radarAngle` מהסירה)
- ג'ויסטיק וירטואלי לשליטה בניווט (ידני)
- סוויץ' מצב ידני/אוטונומי
- כפתור משיכת רשת וכפתור שחרור רשת (לחיצה ממושכת)
- בחירת פורט COM של ארדואינו החוף מתוך ה-UI
- מצב `מוק דאטא` דינמי לבדיקות גם בלי חומרה מחוברת

## הרצה

1. חבר את ארדואינו החוף למחשב.
2. הפעל את השרת המקומי מהתיקייה הראשית של הפרויקט:

```bash
python host/control_server.py
```

3. פתח בדפדפן במחשב את `http://localhost:8765`.
4. בחר ב-UI את פורט ה-COM של החוף ולחץ `חבר לבקר`.

## מצב מוק דאטא

אפשר להדליק את המתג `מוק דאטא` בראש המסך כדי לעבוד עם טלמטריה דינמית סימולטיבית.
במצב זה אין צורך בחיבור Serial, ועדיין אפשר לבדוק:

- תצוגת מכ"ם בזמן אמת
- שליטה ידנית ואוטונומית
- כפתורי כננת

## גישה מהטלפון

1. השאר את השרת רץ על המחשב.
2. חבר את הטלפון למחשב כך שתהיה תקשורת רשת ביניהם.
3. פתח בטלפון את `http://<computer-ip>:8765`.

בפועל זה יכול לעבוד בכמה דרכים:

- אותו Wi-Fi של הטלפון והמחשב
- USB tethering / USB networking מהטלפון למחשב

אם הטלפון מחובר רק לטעינה, בלי ערוץ רשת, לא תהיה גישה ל-UI.

## בדיקה מהירה

```bash
curl http://localhost:8765/api/state
```

## הערות חשובות

- אם מתקבלת הודעת שגיאה שאינה CSV, היא תוצג ב-UI כהודעת שרת.
- בעת ניתוק נשלחת פקודת עצירה: `0,0,0,90`.
- ייתכן שתצטרך לאשר את פורט `8765` ב-Windows Firewall כדי שהטלפון יוכל להגיע לשרת.
