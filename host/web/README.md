# שו"ב בדפדפן

ממשק שו"ב זה עובד דרך Web Serial ומדבר ישירות עם ארדואינו החוף בפרוטוקול שמוגדר ב-API.md.

## מה כולל הממשק

- תמונת מכ"ם ב-Canvas (מבוססת טלמטריה `usRadar` + `radarAngle` מהסירה)
- ג'ויסטיק וירטואלי לשליטה בניווט (ידני)
- סוויץ' מצב ידני/אוטונומי
- כפתור משיכת רשת וכפתור שחרור רשת (לחיצה ממושכת)

## הרצה

1. חבר את ארדואינו החוף למחשב.
2. הפעל שרת HTTP מקומי מתוך התיקייה `host/web`.
3. פתח בדפדפן Chrome או Edge את הכתובת של השרת.
4. לחץ `התחבר ל-Serial` ובחר את ה-COM של החוף.

דוגמה לשרת מקומי מהטרמינל (נדרש Node.js):

```bash
cd host/web
npx serve .
```

## הערות חשובות

- Web Serial דורש הקשר מאובטח (HTTPS או localhost).
- אם מתקבלת הודעת שגיאה שאינה CSV, היא תירשם לקונסול הדפדפן.
- בעת ניתוק נשלחת פקודת עצירה: `0,0,0,90`.
