# מדריך הפעלה — סירה אוטונומית

מסמך זה מסביר **תכל'ס** איך מריצים את השרת, איך פותחים את האתר, ואיך מפעילים את המערכת.

---

## מה צריך לפני שמתחילים

- מחשב עם **Python 3** מותקן.
- הספרייה `pyserial` מותקנת:
  ```bash
  pip install pyserial
  ```
- ארדואינו **החוף** (shore) מחובר למחשב ב-USB.
  - אם עדיין לא צרבת את הקוד לארדואינו, ראה [צריבת הארדואינו](#צריבת-הארדואינו).
- (אופציונלי) ארדואינו **הסירה** (boat) מופעל ומשדר.

> אפשר להריץ את כל הממשק **גם בלי חומרה** בעזרת מצב `מוק דאטא` (ראה למטה).

---

## שלב 1 — הרצת השרת

מהתיקייה הראשית של הפרויקט הרץ:

```bash
python host/control_server.py
```

אמורה להופיע השורה:

```
Serving control UI on http://0.0.0.0:8765
```

זהו — השרת רץ. השאר את החלון הזה פתוח כל עוד אתה עובד.

### אפשרויות (לא חובה)

| דגל        | ברירת מחדל | תיאור                                   |
| ---------- | ---------- | --------------------------------------- |
| `--port`   | `8765`     | פורט ה-HTTP שהשרת מאזין עליו             |
| `--host`   | `0.0.0.0`  | כתובת ההאזנה (`0.0.0.0` = פתוח גם לרשת)  |

דוגמה להרצה בפורט אחר:

```bash
python host/control_server.py --port 9000
```

## שלב 2 — פתיחת האתר

### מהמחשב שמריץ את השרת

פתח בדפדפן:

```
http://localhost:8765
```

### מהטלפון (או מחשב אחר באותה רשת)

1. ודא שהשרת ממשיך לרוץ על המחשב.
2. מצא את כתובת ה-IP של המחשב ברשת (למשל `192.168.1.20`).
3. פתח בטלפון:
   ```
   http://<כתובת-ה-IP-של-המחשב>:8765
   ```
   לדוגמה: `http://192.168.1.20:8765`

> **חשוב:** הטלפון והמחשב חייבים להיות באותה רשת (אותו Wi-Fi, או USB tethering).
> טלפון שמחובר רק לטעינה — לא יקבל גישה.
>
> ב-Windows ייתכן שתצטרך לאשר את פורט `8765` ב-Firewall כדי שהטלפון יגיע לשרת.

## שלב 3 — הפעלה בפועל

1. באתר, בחר מתוך הרשימה את **פורט ה-COM** של ארדואינו החוף.
2. לחץ **`חבר לבקר`**. אמור להופיע חיווי שהחיבור פעיל.
3. עכשיו אפשר לשלוט:
   - **ג'ויסטיק** — שליטה ידנית בניווט (קדימה/אחורה/פניות).
   - **מתג ידני/אוטונומי** — מעבר בין שליטה ידנית ללוגיקת ניווט אוטונומית.
   - **משיכת רשת / שחרור רשת** — לחיצה ממושכת על הכפתור מפעילה את הכננת.
   - **תצוגת מכ"ם** — מציגה בזמן אמת את נתוני החיישן המסתובב.

### לניתוק

לחץ על כפתור הניתוק ב-UI. השרת ישלח אוטומטית פקודת עצירה מלאה (`0,0,0,90`).

---

## מצב מוק דאטא (בדיקה בלי חומרה)

רוצה לבדוק את הממשק בלי לחבר ארדואינו?

1. הרץ את השרת ופתח את האתר כרגיל.
2. הדלק את המתג **`מוק דאטא`** בראש המסך.

במצב זה השרת מייצר טלמטריה סימולטיבית, ואפשר לבדוק:
- תצוגת מכ"ם בזמן אמת
- שליטה ידנית ואוטונומית
- כפתורי הכננת

## בדיקה מהירה שהשרת חי

```bash
curl http://localhost:8765/api/state
```

אם חוזר JSON עם מצב המערכת — השרת עובד תקין.

---

## צריבת הארדואינו

צריבה נעשית עם [PlatformIO](https://platformio.org/). ודא ש-`upload_port`
בקובץ [platformio.ini](platformio.ini) מצביע על פורט ה-COM הנכון של כל ארדואינו.

```bash
# צריבת ארדואינו החוף (מחובר למחשב)
pio run -e shore -t upload

# צריבת ארדואינו הסירה
pio run -e boat -t upload
```

## פתרון תקלות

| תסמין                                   | מה לבדוק                                                        |
| --------------------------------------- | -------------------------------------------------------------- |
| האתר לא נפתח מהמחשב                      | ודא שהשרת רץ ושהחלון עם `Serving control UI...` פתוח            |
| האתר לא נפתח מהטלפון                     | אותה רשת? כתובת IP נכונה? פורט `8765` מאושר ב-Firewall?         |
| אין פורט COM ברשימה                      | ודא שהארדואינו מחובר ושהדרייבר מותקן; נסה כבל USB אחר           |
| `ModuleNotFoundError: serial`           | הרץ `pip install pyserial`                                     |
| החיבור נכשל / הודעת שגיאה ב-UI           | ודא שאין תוכנה אחרת (כמו Serial Monitor) שמחזיקה את הפורט       |
